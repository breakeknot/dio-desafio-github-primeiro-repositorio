Comandos para o Windows:
          - CD: navegar/voltar entre pastas/arquivos

          - MKDIR: criar pasta
          - DEL:  deletar arquivo
          - RMDIR: deletar repositorio/pasta e arquivos
          - CLS : limpar o terminar windows
          - Botão TABLE = autocompletar
